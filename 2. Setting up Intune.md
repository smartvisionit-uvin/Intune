In this lab, I will be creating an Intune Enrollment profile and enrolling a virtual machine.


First, make sure your user account has the required licenses. Minimum a Microsoft Premium license with Intune enabled.

Next, create a group for your Intune-enrolled devices


Go to admin.intune.com. On the left pane select Groups and create new group.


<img width="866" height="499" alt="image" src="https://github.com/user-attachments/assets/eb5fa0b8-4bcb-4b4f-b16a-ab7f276d55c1" />


Go to admin.intune.com. On the left pane select Devices, Enrollment, and set MDM user scope to All
